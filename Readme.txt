To run the code in your local machine 
Step1 - Clone the files into your local repository

Go to the respective folder in your local repository, in the command prompt run the code shown below:
git clone https://github.com/Shabarish033/MapDataBuildingIdentification

Step2 - Clone the Mask RCNN files from Matterport into your local repository, in the command promp run the code shown below:
git clone https://github.com/matterport/Mask_RCNN

Step3 - Unzip the 'building.rar' file from 'MapDataBuildingIdentification' clone

Step4 - Run the "BuildingIdentificationUsingRCnn.ipynb" file on a Jupyter Notebook
